# 🖼️ K-Logo Gen
사용자가 원하는 Prompt 입력을 통해서 로고를 제작해주는 프로그램입니다.
사업분야나 색 표현, 피사체 등을 Prompt에 입력하여 4가지 모델을 통해 생성된 로고를 체험할 수 있습니다.
지금 서비스 주소를 통해 원하는 로고를 만들어 보세요!

서비스 주소 : http://www.klogogen.studio
